> **把時間做成積木，讓計畫像拼圖一樣自由。**  
> *A visual, resource-allocation-based weekly time-blocking planner for web and mobile.*

## 📌 About (關於專案)

**Assemblage**（立體拼貼/組合作品）是一個極簡、直覺且具備「資源限額配給」概念的時間積木排程工具。

有別於傳統繁瑣的行事曆應用，Assemblage 將時間區段抽象化為可拖曳的實體積木。你可以為不同科目或任務設定時間長度與「數量限額」，將它們精確吸附在每週的時間畫布上。當配額耗盡，積木將進入灰底冷卻；當你清除或回收計畫，資源將自動補回。

---

## ✨ Features (核心特色)

* 🧩 **自由時間畫布**：支援 24 小時網格，拖曳放開自動對齊整點（Grid Snapping）。
* 📦 **配給制庫存管理**：動態調整積木可用數量，用完自動灰底鎖定，拖回倉庫或刪除自動回收配額。
* 🎨 **色彩視覺編碼**：自動根據時間長度（1h / 2h / 3h）進行視覺色彩區隔，清晰明瞭。
* 📱 **全平台觸控支援**：
  * **Desktop**：完整支援 HTML5 Drag & Drop 體驗。
  * **Mobile**：獨家 Touch 手勢引擎 + 單日切換分頁（Tabs），手機操作流暢不卡頓。
* 💾 **無障礙數據同步**：
  * 本地自動存檔 (`LocalStorage`)，重新整理不丟失。
  * 一鍵生成 Base64 密鑰，輕鬆跨裝置匯入/匯出課表。
* 🧹 **一鍵重置排列**：一鍵清空畫布，並精確將所有時數回收至庫存。

## 🚀 Quick Start (快速開始)

本專案為純前端無後端架構，無需安裝任何依賴套件。

1. **線上直接體驗**：造訪 [Assemblage Live Demo](https://fuuhakugetsurei.github.io/Assemblage/)
2. **本地執行**：
   ```bash
   git clone https://github.com/fuuhakugetsurei/Assemblage.git
   cd time-block-schedule
   # 直接使用瀏覽器開啟 index.html 即可

   ```
